# Spring Boot + Angular 15 + MySQL CRUD example

Full-stack Angular 15 + Spring Boot + MySQL 

##RUN ECLIPSE/INTELLI

## Run Spring Boot application
```
mvn spring-boot:run
```
The Spring Boot Server will export API at port `8081`.

## Run Angular Client
```
npm install
ng serve --port 8081
```
